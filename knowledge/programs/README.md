# Programs / Majors

Put major- or program-specific laptop requirements here (e.g., Nursing, Engineering, Architecture).

## What goes here

- Program/major overviews (what students typically do on a laptop)
- Common required software (what the school requires; include links)
- “Usually fine on Mac” vs “Windows strongly recommended” guidance
- What follow-up questions the bot should ask first

## File naming

- `knowledge/programs/nursing.md`
- `knowledge/programs/engineering.md`
- `knowledge/programs/architecture.md`
- School-specific variants if needed:
  - `knowledge/programs/<school>/nursing.md`

## Suggested structure (copy/paste)

```md
# Program: <Name>

## Summary
- (1–3 bullets)

## Common software
- <Software A> (OS requirement, notes)
- <Software B> (OS requirement, notes)

## Recommended approach
- If you don’t have the school’s required software list yet, ask for it.
- If proctoring/exam software is required, prioritize compatibility over raw performance.

## Sources
- <school requirement page>
```
