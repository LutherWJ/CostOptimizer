# Policies

Put purchasing/IT/security policy docs here (allowed vendors, required security software, warranty rules).

## What goes here

- School/company IT requirements (VPN, antivirus/EDR, encryption, device management)
- Purchasing rules (warranty length, accidental damage coverage, budget limits)
- Exam policies (proctoring requirements, webcam/mic rules, admin-rights constraints)

## Suggested structure (copy/paste)

```md
# Policy: <Name>

## Summary
- (1–5 bullets)

## Details
### Requirements
- (example) “Windows 11 Pro required”
- (example) “Must support full-disk encryption”

### Exceptions / Workarounds
- (example) “If macOS is allowed, remote Windows desktop is acceptable”

## Sources
- <official policy link>
```
