# Guides

These docs are “how to decide” guidance that helps the bot answer questions like:
- “Should I get 16GB RAM if the minimum says 8GB?”
- “With a $1000 budget, what should I prioritize?”
- “Is Mac or Windows safer for college software compatibility?”

Keep guides short and written in plain language.
