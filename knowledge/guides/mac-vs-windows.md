# Mac vs Windows (College Compatibility Guide)

Last reviewed: 2026-04-18  
Status: draft - replace with school-specific requirements when available.

## The simple rule

- If your school/program requires Windows-only software or strict exam/proctoring tools, Windows is usually the safer choice.
- If your school explicitly supports macOS for all required tools, a Mac can be fine.

## Why exam/proctoring software matters

Some exam tools have strict rules and can break with:
- virtual machines (running Windows inside macOS)
- remote desktop/VDI
- screen recording/sharing software

So even if a workaround exists, it may not be acceptable for exams.

## Best follow-up questions

1) “What exam/proctoring software does your program require (exact name)?”
2) “Does the school explicitly allow macOS for exams?”
3) “Do they forbid VMs or remote desktop for exams?”
